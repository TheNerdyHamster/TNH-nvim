# TNH-Nvim for the Hamsters!

## Languages supported
  - Golang
  - Rust
  - Lua
  - PHP - Ongoing

## Installation
### Requierd packages
- lua
- ninja

First run `:PackerInstall` after that run `:luafile ~/.local/share/nvim/site/pack/packer/start/nlua.nvim/scripts/download_sumneko.lua` to install packages and sumneko
